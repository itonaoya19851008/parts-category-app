# 部品カテゴリ分類・集計アプリ

製造業の出図管理業務を効率化するためのStreamlitアプリケーションです。
部品表（ALL_PARTS_LIST）のデータを読み込み、部品をカテゴリ分けして集計します。

## 機能

- Excel/CSV形式の部品表ファイルをアップロード
- 各部品（PARTS単位）を「d, Ds, Dm, De, PD」の5カテゴリに自動分類
- Itemごとにカテゴリ別の「合計部品数」と「合計重量（重量×数量）」を集計
- 集計結果をプレビュー表示
- 2段ヘッダー形式のExcelファイル（.xlsx）としてダウンロード可能

## セットアップ

### 必要な環境

- Python 3.8以上
- pip

### インストール

```bash
pip install -r requirements.txt
```

### アプリの起動

```bash
streamlit run app.py
```

ブラウザが自動的に開き、アプリケーションが表示されます。

## 使い方

1. **ファイルのアップロード**
   - 画面上部のファイルアップローダーから、部品表ファイル（.xlsx, .xls, .csv）を選択

2. **データの確認**
   - 読み込んだデータのプレビューを確認できます

3. **分類・集計の実行**
   - 「🔍 分類・集計を実行」ボタンをクリック
   - 処理が完了すると、集計結果が表示されます

4. **結果のダウンロード**
   - 「📥 集計結果をExcel形式でダウンロード」ボタンから、Excelファイルをダウンロードできます

## 分類ロジック

部品は以下の12段階の条件に基づいて分類されます：

1. SUMMARY欄に「MS-」または「JIS」を含む → 定義対象外（スキップ）
2. SUPPLY欄に「ES, SU, ST, CS」を含むか？
3. NAME OF PARTSに「Roll」または「Roller」を含むか？
4. NAME OF PARTSに「BRG」または「Bearing」を含むか？
5. 部品単品重量が3000kg以上か？ → **Ds**
6. MATERIALに「SPCC, SPHC, SS*」を含むか？
7. 構成部材に「SPCC, SPHC, SS*以外の材質」で重量100kg以上のものが含まれるか？ → **Dm**
8. 構成部材のSIZE等に80mm厚以上（t80以上など）が使用されているか？
9. 部品単品重量が500kg以上か？ → **Dm**
10. PARTSまたはPIECEの名称に「PIPE」または「PIPING」があるか？ → **PD**
11. 部品単品重量が50kg未満か？ → **PD**
12. 残部品は半数ずつ「Dm」と「De」に按分

## 出力フォーマット

Excel出力ファイルは以下の形式です：

- **1段目ヘッダー**: Order名, Order＃, Item＃, Item名称, d, d, Ds, Ds, Dm, Dm, De, De, PD, PD
- **2段目ヘッダー**: (空白), (空白), (空白), (空白), 部品数, 重量, 部品数, 重量, 部品数, 重量, 部品数, 重量, 部品数, 重量

## 注意事項

- Excelファイルは特殊な構造（1-4行目にORDER/ITEM情報、その下にメインテーブル）に対応しています
- 列名にセル内改行や空白が含まれていても自動的に処理されます
- PARTSとPIECEの階層構造を考慮して分類・集計を行います

## ライセンス

このプロジェクトは内部使用を目的としています。
