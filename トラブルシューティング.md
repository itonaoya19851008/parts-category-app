# トラブルシューティング

## 問題：`python -m pip install` を実行しても「Python」とだけ表示される

### 原因
Windows StoreのPythonランチャーが優先されている可能性があります。

### 解決方法

#### 方法1：Pythonの実体を確認する

PowerShellで以下を実行：

```powershell
# Pythonの場所を確認
Get-Command python | Select-Object -ExpandProperty Source

# Pythonのバージョンを確認
python --version

# もしエラーが出る場合、直接パスを指定
& "C:\Python3x\python.exe" -m pip install -r requirements.txt
```

#### 方法2：Pythonを再インストール

1. https://www.python.org/downloads/ からPythonをダウンロード
2. インストール時に「Add Python to PATH」にチェックを入れる
3. インストール後、PowerShellを再起動

#### 方法3：仮想環境を使用

```powershell
# 仮想環境を作成
python -m venv venv

# 仮想環境を有効化
.\venv\Scripts\Activate.ps1

# パッケージをインストール
pip install -r requirements.txt

# アプリを起動
streamlit run app.py
```

#### 方法4：Anaconda/Minicondaを使用

もしAnacondaがインストールされている場合：

```powershell
conda install streamlit pandas numpy openpyxl
streamlit run app.py
```

### 確認コマンド

PowerShellで以下を実行して、Pythonが正しく動作するか確認：

```powershell
# Pythonが実行できるか確認
python -c "print('Hello, Python!')"

# pipが使えるか確認
python -m pip --version

# インストール済みパッケージを確認
python -m pip list
```

### エラーメッセージの確認

もしエラーメッセージが出ている場合、その内容を確認してください：

```powershell
# 詳細なエラー情報を表示
python -m pip install -r requirements.txt --verbose
```
